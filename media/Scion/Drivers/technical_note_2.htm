Windows 7 Support for Scion FireWire Cameras
 
 
 
 
All Scion FireWire (IEEE-1394) cameras support the Windows 7 Operating System.  However, it is currently necessary to downgrade the Windows FireWire drivers to the Legacy version. 

These steps are not necessary if the following is true: 
CFW-1000 Series cameras with serial numbers higher than 10000012
CFW-1300 Series cameras with serial numbers higher than 13006722 
CFW-1600 Series cameras with serial numbers higher than 16000161

  1. Download and install the Windows Vista drivers located on the Scion website.
  2. Disconnect the camera from the computer.
  3. Click the Start button and right-click the “Computer” entry.  On the menu select “Manage”.  The Computer Management window will appear.
  4. On the left pane select the “Device Manager” entry.
  5. In the center pane find the “IEEE 1394 Bus host controllers” entry and click the arrow to the left.  The controller will appear below.
  6. Right click the controller and select “Update Driver Software”.

  1. Select the “Browse my computer for driver software” button in the new window.
  2. Next select the “Let me pick from a list of device drivers on my computer” button.
  3. In the list that appears select the “1394 OHCI Compliant Host controller (Legacy)” and click the Next button.

  1. The wizard should show a successful installation of the driver.  Click the Close button.
  2. Now connect the Scion camera.  The Device Manager will automatically update showing the Scion camera model just below “Other Devices”.
  3. Right click the camera model just under the “Other Devices” and select “Update Driver Software”.
  4. In the new window select “Browse my computer for driver software”.
  5. On this new page click the Browse button.  Browse out to “C:\Program Files\Scion Corporation\1394 Drivers” and click the OK button.  Click Next to continue.
  6. A window will appear alerting the user that it cannot verify the publisher of the software.  Select “Install this driver software anyway”.
  7. A new window will appear after a short time showing that the driver was installed successfully.  Click the Close button to finish the driver installation.
